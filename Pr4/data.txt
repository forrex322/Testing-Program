Form-це представлення будь-якого вікна, що відображається в додатку. Клас Form можна використовувати для створення стандартних вікон, інструментів, без кордонів і плаваючих вікна. Можна також використовувати клас Form для створення модальних вікон, таких як діалогове вікно. Особливий вид форми, форма багатодокументного інтерфейсу (MDI), може містити інші форми, звані дочірніми формами MDI. Форма MDI створюється шляхом присвоєння властивості IsMdiContainer значення true. Дочірні форми MDI створюються шляхом присвоєння властивості mdiparent батьківської MDI-форми, яка буде містити дочірню форму.
Використовуючи властивості, доступні в класі Form, можна визначити зовнішній вигляд, розмір, колір і функції управління вікнами створюваного вікна або діалогового вікна. Властивість Text дозволяє вказати заголовок вікна в рядку заголовка. Властивості Size і DesktopLocation дозволяють визначити розмір і розташування вікна при його відображенні. Властивість Колір ForeColor можна використовувати для зміни кольору переднього плану за замовчуванням для всіх елементів управління, розміщених у формі. Властивості FormBorderStyle, Minimizeboxі MaximizeBox дозволяють управляти можливістю згортання, розгортанням або зміною розміру форми під час виконання.
На додаток до властивостей можна використовувати методи класу для управління формою. Наприклад, можна використовувати метод ShowDialog для відображення форми у вигляді модального діалогового вікна. Для розміщення форми на робочому столі можна використовувати метод SetDesktopLocation.
Події класу Form дозволяють реагувати на дії, що виконуються у формі. Подія Activated можна використовувати для виконання таких операцій, як оновлення даних, що відображаються в елементах управління форми при активації форми.
Форму можна використовувати в якості початкового класу програми, помістивши в клас метод з ім'ям Main. У методі Main додайте код для створення та відображення форми. Крім того, необхідно додати атрибут STAThread в метод Main, щоб форма була запущена. Коли початкова форма закривається, додаток також закривається.
Якщо для властивості Enabled задано значення false перед відображенням Form (наприклад, при установці Enabled в значення false в конструкторі Microsoft Visual Studio), кнопки згортання, розгорнути, закрити і система залишаються включеними. Якщо для параметра Enabled задано значення false після того, як form відображається (наприклад, при виникненні події Load), кнопки будуть відключені.
